# Exercícios de Python — PUC Minas

Repositório com exercícios de Python resolvidos durante o **1º período** do curso de Engenharia de Software na PUC Minas (Coração Eucarístico).

## 📂 Organização

Os exercícios estão separados por tema/tópico estudado, em ordem crescente de complexidade:

| Pasta | Conteúdo |
|---|---|
| `01-fundamentos` | Variáveis, tipos de dados, entrada e saída |
| `02-estruturas-de-controle` | Condicionais (`if`/`else`) e laços de repetição (`for`/`while`) |
| `03-funcoes` | Definição e uso de funções, parâmetros e retorno |
| `04-colecoes` | Listas, tuplas, dicionários e conjuntos |
| `05-recursividade` | Funções recursivas |
| `06-orientacao-a-objetos` | Classes, objetos, atributos e métodos |
| `07-tratamento-de-excecao` | Tratamento de erros com `try`/`except` |
| `08-manipulacao-de-arquivos` | Leitura e escrita de arquivos |

## 🛠️ Tecnologias

- Python 3

## ▶️ Como executar

Cada exercício é um arquivo `.py` independente. Para rodar:

```bash
python nome_do_arquivo.py
```
